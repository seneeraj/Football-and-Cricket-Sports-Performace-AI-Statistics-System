# ⚽🏏 Multi-Sport AI Analytics Dashboard

An AI-powered analytics dashboard for **Football (EPL)** and **Cricket (IPL)** that predicts player performance and provides deep visual insights using Machine Learning.

---

## 🚀 Project Overview

This project combines **Machine Learning + Sports Analytics + Interactive Dashboarding** to:

- Predict player performance scores  
- Compare players side-by-side  
- Analyze player strengths vs average  
- Provide ranking and percentile insights  
- Deliver visual analytics using real-world datasets  

---

## 🎯 Key Features

### ⚽ Football Analytics
- Player selection (EPL dataset)
- Performance prediction using Linear Regression
- KPI metrics (Goals, Assists, Influence)
- Player vs Average comparison
- Performance trend visualization
- Top players leaderboard
- Goal distribution analysis
- Player ranking & percentile system
- Player vs Player comparison

---

### 🏏 Cricket Analytics
- Player selection (IPL dataset)
- Impact score prediction
- KPI metrics (Runs, Strike Rate, Wickets)
- Player vs Average comparison
- Performance trend visualization
- Top players leaderboard
- Runs distribution analysis
- Player ranking & percentile system
- Player vs Player comparison

---

## 🧠 Machine Learning

### Models Used:
- **Multiple Linear Regression (OLS)**

### Football Model Inputs:
- Goals, Assists, Minutes
- Influence, Creativity, Threat
- ICT Index, Form, BPS
- Clean Sheets, Goals Conceded

### Cricket Model Inputs:
- Runs
- Strike Rate
- Wickets
- Fours
- Sixes
